The Embassy Components

On this website you can find the assets for The Embassy of Good Science based on Bootstrap 4.

These will be used on a MediaWiki skin (Chameleon) for The Embassy platform.

Be aware that not all Bootstrap funcionality is in this files. Only the necessary usage on the platform.

You can download these here:
  • main.css - all styles for bootstrap and custom component
  • vendors.js - all libraries used and bootstrap javascript components
  • main.js - all other custom js modules used in the platform
