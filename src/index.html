The Embassy Components

On this website you can find the thematic assets styles of Bootstrap 4 to be used on a Chameleon skin used in The Embassy of Good Science platform.

The assets that set all Bootstrap definition and javascript funcionality as follow:
  • main.css - all styles for bootstrap and custom component
  • vendors.js - all libraries used and bootstrap javascript components
  • main.js - all other custom js modules used in the platform
